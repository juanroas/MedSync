# Fluxo

1. Ler a visao de produto.
2. Ler o posicionamento do produto.
3. Consultar referencias de mercado.
4. Atualizar a matriz de rastreabilidade de referencias.
5. Ler o escopo candidato do MVP.
6. Ler o modelo de atores e organizacoes.
7. Ler as diretrizes de CRUD e atualizacao de dados.
8. Ler o modelo de negocio.
9. Consultar personas e jornadas.
10. Revisar catalogo de funcionalidades.
11. Consultar arquitetura de informacao.
12. Revisar roadmap e modelo B2B.
13. Registrar lacunas como `TODO` em documentos apropriados.

# Boas práticas

- Nao inventar regras de negocio.
- Nao tratar referencia externa como incorporada sem rastreabilidade e backlog.
- Nao declarar conformidade regulatoria sem aprovacao formal.
- Linkar especificacoes quando forem preenchidas.
