# Copywriting Guidelines - MedSync

Status: Blueprint. Textos finais precisam de validacao de produto, juridico e privacidade.

## Objetivo

Definir diretrizes de linguagem para o MedSync.

## Escopo

Inclui tom, mensagens permitidas, mensagens proibidas, exemplos e cuidados regulatórios.

## Tom de voz

- Claro.
- Sobrio.
- Confiavel.
- Profissional.
- Orientado a acao.
- Cuidadoso com regulacao e privacidade.

## Nao usar

- "100% seguro".
- "Em conformidade com LGPD".
- "Aprovado pelo CFM".
- "Pronto para pacientes reais".
- "Diagnostico garantido".
- "Substitui consulta presencial".

## Usar com cautela

- "Homologacao controlada".
- "Privacidade por design".
- "Relatorios agregados".
- "Controles de acesso".
- "Evidencias de auditoria".

## Mensagens de erro

Diretrizes:

- Explicar o que ocorreu.
- Orientar proximo passo.
- Nao expor detalhes tecnicos sensiveis.
- Nao confirmar existencia de usuario em login.
