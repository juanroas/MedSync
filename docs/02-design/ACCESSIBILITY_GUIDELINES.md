# Accessibility Guidelines - MedSync

Status: Blueprint.

## Objetivo

Definir requisitos iniciais de acessibilidade para a futura interface do MedSync.

## Escopo

Inclui diretrizes para contraste, foco, teclado, texto, formularios e feedback.

## Diretrizes

- Todo elemento interativo deve ter foco visivel.
- Campos devem ter labels claros.
- Erros devem indicar problema e acao esperada.
- Cor nao deve ser unico indicador de significado.
- Texto deve ser legivel em desktop e mobile.
- Modais devem controlar foco quando implementados.
- Tabelas devem ser navegaveis e compreensiveis.
