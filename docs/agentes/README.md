# Objetivo

Organizar regras, agentes, prompts, workflows e templates para uso de IA no MedSync.

# Escopo

Inclui regras gerais, perfis de agentes, prompts por especialidade, workflows e templates de entrega.

# Fluxo

1. Ler `AI_AGENT_RULES.md`.
2. Ler `EXECUTION_GUARDRAILS.md`.
3. Ler `SKILLS_STANDARD.md` quando criar ou atualizar skills.
4. Escolher o agente ou prompt adequado.
5. Seguir o workflow relacionado.
6. Usar templates quando houver entrega documental ou tecnica.

# Boas práticas

- Nao extrapolar escopo.
- Nao confundir documentacao com implementacao.
- Nao concluir tarefas baseadas em referencias externas sem rastreabilidade.
- Nao inventar regras, APIs ou tabelas.
- Registrar lacunas como `TODO`.
