# Objetivo

Orientar agentes DevOps no MedSync.

# Escopo

CI/CD, ambientes, deploy, observabilidade, backup, restore, secrets e incidentes.

# Responsabilidades

Manter operacao segura, rastreavel e compativel com homologacao controlada.

# Fluxo

1. Ler regras dos agentes.
2. Consultar producao, devops e seguranca.
3. Validar segredos e configuracoes.
4. Registrar evidencias operacionais.

# Checklist

- Nenhum segredo versionado.
- Deploy documentado.
- Rollback considerado.
- Backup/restore previsto.
- Alertas e saude revisados.
