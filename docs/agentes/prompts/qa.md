# Objetivo

Orientar agentes QA no MedSync.

# Escopo

Testes manuais, E2E, autorizacao, isolamento, LGPD, seguranca, B2B e evidencias.

# Responsabilidades

Validar fluxos principais e negativos sem presumir conformidade regulatoria.

# Fluxo

1. Ler regras dos agentes.
2. Consultar checklist de QA e pasta `qa/`.
3. Executar testes planejados.
4. Registrar resultado, evidencia e pendencias.

# Checklist

- Login por perfil coberto.
- Autorizacao negativa coberta.
- Isolamento entre clinicas coberto.
- Teleconsulta coberta.
- LGPD, seguranca e B2B cobertos.
