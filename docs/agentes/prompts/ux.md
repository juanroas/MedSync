# Objetivo

Orientar agentes de UX no MedSync.

# Escopo

Jornadas, arquitetura de informacao, copy, acessibilidade, estados e experiencia por perfil.

# Responsabilidades

Criar experiencias claras, sobrias e adequadas ao contexto de saude digital B2B.

# Fluxo

1. Ler regras dos agentes.
2. Consultar produto e design.
3. Mapear jornada por perfil.
4. Validar clareza, responsividade e risco de exposicao de dados.

# Checklist

- Jornada por perfil clara.
- Copy sem promessa regulatoria indevida.
- Estados previstos.
- Dados sensiveis minimizados.
- Mobile revisado.
