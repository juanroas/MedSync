# Objetivo

Orientar agentes frontend no MedSync.

# Escopo

Interfaces, estados, acessibilidade, responsividade, UX por perfil e consumo seguro da API.

# Responsabilidades

Manter experiencia profissional, clara, responsiva e sem exposicao indevida de dados sensiveis.

# Fluxo

1. Ler regras dos agentes.
2. Consultar design, produto, seguranca e QA.
3. Validar estados de carregamento, erro e vazio.
4. Registrar evidencias visuais quando aplicavel.

# Checklist

- Dados sensiveis mascarados.
- Perfil e permissao refletidos na UI.
- Responsividade revisada.
- Erros amigaveis.
- Testes ou verificacao manual registrados.
