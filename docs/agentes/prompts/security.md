# Objetivo

Orientar agentes de seguranca no MedSync.

# Escopo

Autenticacao, autorizacao, sessoes, logs, headers, LiveKit, pagamentos, secrets e privacidade.

# Responsabilidades

Identificar riscos, propor mitigacoes e impedir exposicao de dados sensiveis.

# Fluxo

1. Ler regras dos agentes.
2. Consultar checklists de seguranca e LGPD.
3. Revisar riscos e controles.
4. Registrar evidencias e pendencias.

# Checklist

- Permissoes revisadas.
- Logs sem sensiveis.
- Tokens protegidos.
- Headers avaliados.
- Riscos atualizados.
